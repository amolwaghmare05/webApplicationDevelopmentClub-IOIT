# Team Member Photos

Place team member photos in this folder with the following naming convention:

## File Naming Format
Use the team member's name in lowercase with hyphens for spaces:
- `swaroop.jpg`
- `ashish-kharde.jpg`
- `bhumi-shinde.jpg`
- `sarvesh-kolte.jpg`
- `chinmay-nakwa.jpg`
- `pritam-rangari.jpg`
- `amol-waghmare.jpg`
- `niraj-shevade.jpg`
- `president.jpg`
- `vice-president.jpg`
- `faculty-advisor.jpg`
- `event-head.jpg`
- `event-joint-head.jpg`

## Recommended Specifications
- Format: JPG or PNG
- Size: 400x400 pixels (square)
- Aspect Ratio: 1:1
- File Size: Under 200KB for optimal loading
- Background: Professional, preferably solid or blurred

## Usage
These photos will be automatically displayed in the Team section based on the file name matching the team member's name in the data.
