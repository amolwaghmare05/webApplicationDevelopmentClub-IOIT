# Event Photos Gallery

Place event photos in this folder to showcase club activities.

## Folder Structure
Organize photos by event or category:

```
eventPhotos/
├── workshops/
│   ├── web-dev-bootcamp-1.jpg
│   ├── web-dev-bootcamp-2.jpg
│   └── git-workshop-1.jpg
├── hackathons/
│   ├── code-for-change-1.jpg
│   └── code-for-change-2.jpg
├── tech-talks/
│   ├── ai-webdev-talk-1.jpg
│   └── cloud-computing-1.jpg
├── team/
│   ├── team-photo-2024.jpg
│   └── team-photo-2025.jpg
└── general/
    ├── event-1.jpg
    ├── event-2.jpg
    └── event-3.jpg
```

## File Naming Convention
- Use lowercase with hyphens
- Include event name and number: `event-name-1.jpg`
- Be descriptive: `hackathon-winners-2025.jpg`

## Recommended Specifications
- Format: JPG (preferred) or PNG
- Resolution: 1200x800 pixels (landscape) or 800x1200 (portrait)
- Aspect Ratio: 3:2 or 4:3 preferred
- File Size: Under 500KB for optimal loading
- Quality: High quality but optimized for web

## Adding Photos to Gallery
After adding photos, update the gallery data in:
`client/src/data/galleryData.ts`
